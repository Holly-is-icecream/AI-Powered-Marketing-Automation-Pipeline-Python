# AI-Driven Marketing Automation Pipeline

## Overview
Designed an end-to-end automation workflow to transform raw marketing data into actionable insights.  
This project simulates a real-world business reporting process, reducing manual effort and improving decision-making efficiency.

## Key Capabilities
- Automated data ingestion, cleaning, and KPI calculation (CTR, conversion rate)
- Channel-level performance analysis and reporting
- Rule-based recommendation engine for decision support
- AI-style insight generation for business-friendly summaries
- Automation impact tracking (~85% reduction in manual effort)

## Tech Stack
Python (Pandas, NumPy)

```bash
pip install pandas openpyxl
